# 安装 Node.js 和启动网站

## 方法 1：使用官方安装包（推荐，最简单）

1. **下载 Node.js**：
   - 访问：https://nodejs.org/
   - 点击下载 "LTS" 版本（推荐版本）
   - 下载完成后，双击 `.pkg` 文件安装

2. **验证安装**：
   安装完成后，**重新打开终端**，然后运行：
   ```bash
   node --version
   npm --version
   ```
   如果显示版本号，说明安装成功！

3. **启动网站**：
   ```bash
   cd /Users/jiangyiwu/perosnal_webpage
   npm install
   npm run dev
   ```

---

## 方法 2：使用 Homebrew（如果已经安装了 Homebrew）

如果你已经安装了 Homebrew，可以运行：
```bash
brew install node
```

然后重新打开终端，运行：
```bash
cd /Users/jiangyiwu/perosnal_webpage
npm install
npm run dev
```

---

## 安装完成后

安装完成后，在终端中运行：
```bash
cd /Users/jiangyiwu/perosnal_webpage
npm install
npm run dev
```

服务器启动后，会显示类似 `http://localhost:3000` 的地址，在浏览器中打开即可查看你的网站！
