# 启动开发服务器

要在浏览器中实时查看你的网站，请在终端中运行以下命令：

## 步骤：

1. **安装依赖**（如果还没有安装）：
   ```bash
   npm install
   ```

2. **启动开发服务器**：
   ```bash
   npm run dev
   ```

3. **在浏览器中打开**：
   服务器启动后，通常会显示一个本地地址（例如：`http://localhost:3000`）
   在浏览器中打开这个地址即可实时查看你的网站。

4. **修改并保存**：
   当你修改代码时，浏览器会自动刷新显示最新效果。

## 注意事项：

- 确保已安装 Node.js（如果未安装，请访问 https://nodejs.org/）
- 照片已经放在 `public/assets/profile-photo.jpg`
- 如果需要更换照片，只需替换这个文件即可
